# Python Text Analyzer

A small Python project to analyze text files for word frequency, character count, and sentence statistics.

## Features
- Count words, sentences, and characters
- Find most common words
- Support multiple text files
- Easy-to-use command-line interface

## Folder Structure
- `src/` → Python source code
- `tests/` → Unit tests
- `data/` → Sample text files
- `docs/` → Documentation
- `.venv/` → Virtual environment
